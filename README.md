## 功能特点

### 📖 多词典支持
集成多种权威词典，满足不同查词需求：

| 词典 | 翻译 | 其他 |
|------|------|------|
| 牛津、韦氏、科林斯、剑桥 | 有道、百度、腾讯、DeepL、谷歌 | 维基百科、百度百科 |

### 📝 单词学习
- **单词本**：收藏整理你查询过的单词
- **背单词**：支持多种词库（ CET4/6、考研、雅思、托福 等）
- **学习进度**：追踪你的学习记录

### 🔧 实用工具
- **语法检查**：检查英文语法错误
- **每日一句**：每日推送英文例句
- **快速查词**：通知栏快捷查词

### 🎨 个性化
- **深色模式**：支持跟随系统或手动切换
- **多主题色**：9 种可选主题颜色
- **欢迎界面**：首次使用引导

## 技术栈

- **语言**：Kotlin
- **最低 SDK**：26 (Android 8.0)
- **目标 SDK**：36
- **架构**：MVVM
- **网络**：OkHttp + XHttp2
- **数据库**：SQLite (SQLCipher)
- **异步**：RxJava / RxAndroid
